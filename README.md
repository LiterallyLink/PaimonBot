# PaimonBot

A interactive bot made with DJS based off of the popular MMORPG Genshin Impact.

## Install

1. Install packages via `npm install`
1. Create config in `./config.json`
1. Start bot via `npm run start`


## Config Example

```
{
  "token": "BOT_TOKEN",
  "mongoPath": "MONGO_URL",
  "invite": "INVITE_URL"
}
```
